    Troubleshooting¶

    Common Issues¶

    Post Failed to Publish: - Cause: Account connection expired, platform API issue, or content violation - Solution: - Check account status in Platforms page - Review platform content policies - Retry publishing - Edit content if flagged

    Images Not Displaying: - Cause: Wrong format, file too large, or unsupported dimensions - Solution: - Convert to supported format (JPG, PNG) - Compress image to under 10MB - Resize to recommended dimensions - Re-upload media

    Scheduled Post Didn't Publish: - Cause: Account disconnected, time zone mismatch, or system error - Solution: - Verify account connection - Check time zone settings - Reschedule post - Contact support if recurring

    Character Limit Exceeded: - Cause: Post too long for platform - Solution: - Trim content to fit - Create thread (X/Twitter) - Split into multiple posts - Use image with text
