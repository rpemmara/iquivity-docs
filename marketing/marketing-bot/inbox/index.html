    Marketing Bot - Inbox¶

    Unified messaging interface for managing conversations across WhatsApp and Telegram in one centralized inbox.

    Overview¶

    The Inbox provides a centralized hub for viewing and responding to messages from both WhatsApp and Telegram contacts. Manage all your conversations, track message history, and engage with your audience from a single interface.

    Unified Communication

    All messages from WhatsApp and Telegram appear in one inbox, making it easy to manage customer conversations across platforms.

    Inbox Layout¶

    Main Interface Sections¶

    Left Sidebar - Message List: - Contact names and phone numbers - Platform indicators (WhatsApp/Telegram) - Message timestamps - Unread message badges - Contact avatars

    Center Panel - Conversation View: - Full message thread - Sent and received messages - Media attachments - Message timestamps - Delivery status

    Right Sidebar - Contact Details: - Contact information - Channel type - Status - Creation date - IP address - Contact ID

    Best Practices¶

    Inbox Management¶

    Response Guidelines: - Reply within 1 hour during business hours - Use saved templates for efficiency - Personalize responses - Keep messages concise - Include clear next steps

    Organization Tips: - Archive resolved conversations - Tag contacts appropriately - Star important messages - Pin urgent conversations - Create segments for organization

    Professional Communication: - Use proper grammar - Be friendly and helpful - Set expectations for response time - Thank customers - Confirm resolution

    Troubleshooting¶

    Common Issues¶

    Messages Not Loading: - Cause: Network connectivity, server issue - Solution: Refresh page, check internet connection, retry

    Can't Send Messages: - Cause: Invalid number, platform issue, rate limit - Solution: Verify contact number, check platform status, wait and retry

    Missing Conversations: - Cause: Archived, deleted, filter applied - Solution: Check archived folder, review filters, search by contact

    Notifications Not Working: - Cause: Browser permissions, notification settings - Solution: Enable browser notifications, check inbox settings
