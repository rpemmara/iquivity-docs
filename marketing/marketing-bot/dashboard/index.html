    Marketing Bot - Dashboard¶

    Centralized automation hub for broadcasting messages and managing campaigns across WhatsApp and Telegram.

    Overview¶

    The Marketing Bot provides smart automations for broadcasting messages to your audience instantly on WhatsApp and Telegram. Create campaigns, manage contacts, track engagement, and automate your messaging workflows from a single dashboard.

    Instant Broadcasting

    Send messages to your entire audience across WhatsApp and Telegram with just a few clicks. Perfect for promotions, announcements, and customer engagement.

    Dashboard Overview¶

    Main Features¶

    • Telegram Integration

      Broadcast messages instantly on Telegram

      Send to groups and individual contacts

    • WhatsApp Integration

      Reach your audience on WhatsApp

      Direct messaging and group broadcasts

    • Manage Settings

      Configure bot settings and integrations

      Set up API tokens and webhooks

    Troubleshooting¶

    Common Issues¶

    Messages Not Sending: - Cause: Invalid API credentials, platform restrictions - Solution: Check settings, verify tokens, ensure compliance

    Low Delivery Rates: - Cause: Invalid numbers, blocked contacts, network issues - Solution: Clean contact list, verify numbers, test connectivity

    Campaign Not Scheduling: - Cause: Incorrect date format, past date selected - Solution: Use date picker, verify time zone, check format

    Analytics Not Updating: - Cause: Cache delay, webhook issues - Solution: Refresh page, check webhook configuration, wait for sync
