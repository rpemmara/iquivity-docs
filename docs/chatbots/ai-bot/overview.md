## Interaction Types

### AI Only

**Fully Automated**
- Bot handles all conversations
- No human intervention
- 24/7 availability
- Scales infinitely

**Best For:**
- FAQ handling
- Information distribution
- Simple support queries
- High-volume, low-complexity interactions

### AI + Live Chat

**Hybrid Approach**
- AI handles initial conversation
- Users can request human agent
- "Get More Help" button available
- Smooth handoff process

**Best For:**
- Customer support
- Sales inquiries
- Complex problem solving
- Quality customer service

### Live Chat Only

**Human-Powered**
- Direct connection to human agents
- AI disabled
- Real-time human support
- Traditional live chat

**Best For:**
- VIP customers
- Technical support
- Sensitive conversations
- High-value sales

## Frequently Asked Questions

??? question "Can I use AI Bot on multiple websites?"
    Yes! You can embed the same AI Bot on multiple websites, or create different bots for each site depending on your plan limits.

??? question "How do I handle questions the bot can't answer?"
    Enable the Human Agent feature to allow seamless handoff to your support team. Users can click "Get More Help" to connect with a human agent.

??? question "Can I customize the bot's appearance?"
    Yes! You can customize colors, branding, welcome messages, chat bubble styling, and more to match your brand.

??? question "What languages are supported?"
    AI Bot supports 50+ languages. You can set the primary language and the bot will detect and respond in the user's language.

??? question "Do I need technical skills to set up AI Bot?"
    No! AI Bot is designed for non-technical users. The setup wizard guides you through each step with no coding required.
