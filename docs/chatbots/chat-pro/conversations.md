## Organizing Conversations

### Creating New Conversations

**To Start a New Chat:**

1. Click the **"+" (New Chat)** button
2. Select a chatbot (Default or Expert)
3. Choose AI model
4. Start typing

**When to Create New Chats:**
- Different topics/projects
- Separate work contexts
- Testing different models
- Privacy/organization

### Switching Between Conversations

**To Switch Chats:**

1. Click on any chatbot in the sidebar
2. Previous conversation loads instantly
3. Context from that conversation preserved
4. Continue where you left off

**Benefits:**
- Keep multiple projects organized
- Maintain separate contexts
- Easy reference to past work
- Quick topic switching

### Naming Conversations

**Auto-Generated Titles:**
- AI automatically titles conversations

**Custom Titles:**

3. Save

**Example Titles:**

## Exporting Conversations

### Export Options

**To Export a Conversation:**

1. Open the conversation you want to export
2. Click **More Options (‚ãØ)** at top-right
3. Select **"Export"**
4. Choose format:
   - **Text (.txt)** - Plain text format
   - **PDF (.pdf)** - Formatted document
   - **Markdown (.md)** - Markdown format
   - **JSON** - Raw data format
5. Click **"Download"**

### Export Formats Explained

**Text (.txt)**
```
Conversation Export
Date: December 6, 2025
Chatbot: Default AI Chat Bot
Model: GPT-4o

User: Explain machine learning in simple terms

AI: Machine learning is a type of artificial intelligence
that allows computers to learn and improve from experience
without being explicitly programmed...

User: Can you give me a practical example?

AI: Sure! A great example is email spam filters...
```

**Best For:**
- Simple archiving
- Copying to other apps
- Quick reference
- Email sharing

**PDF (.pdf)**
- Professional formatting
- Headers and structure
- Syntax highlighting for code
- Images preserved
- Ready to print

**Best For:**
- Presentations
- Client deliverables
- Documentation
- Professional sharing
- Archiving

**Markdown (.md)**
- Structured formatting
- Code blocks with syntax highlighting
- Headers and lists formatted
- Compatible with documentation tools

**Best For:**
- Technical documentation
- GitHub/GitLab wikis
- Knowledge bases
- Developer collaboration

**JSON**
- Raw conversation data
- Includes metadata
- Timestamps
- Model information
- Message structure

**Best For:**
- Data analysis
- Custom processing
- API integration
- Backup purposes

### Bulk Export

**Export Multiple Conversations:**

1. Go to conversation management
2. Select multiple chats (checkbox)
3. Click **"Export Selected"**
4. Choose format
5. Download as ZIP file

**Use Cases:**
- Weekly/monthly backups
- Project archives
- Team knowledge sharing
- Compliance/record keeping

### Managing Shared Conversations

**Permissions Levels:**

**View Only:**
- Can read entire conversation
- Cannot add messages
- Cannot export
- Cannot reshare

**Can Comment:**
- View full conversation
- Add comments/notes
- Cannot modify messages
- Can export

**Can Edit:**
- Full access to conversation
- Can add messages
- Can continue chat
- Can export and reshare

## Conversation Management

### Archiving Conversations

**To Archive:**

1. Right-click on conversation (or use menu)
2. Select **"Archive"**
3. Conversation moves to archive

**Archived Conversations:**
- Removed from main list
- Still searchable
- Can be restored
- Preserved indefinitely

**When to Archive:**
- Completed projects
- Old conversations
- Seasonal content
- Reference material

### Deleting Conversations

**To Delete:**

1. Right-click on conversation
2. Select **"Delete"**
3. Confirm deletion

!!! warning "Deletion is Permanent"
    Deleted conversations cannot be recovered. Export important chats before deleting.

**When to Delete:**
- Test conversations
- Duplicate chats
- Sensitive information (after use)
- Temporary experiments

### Restoring Conversations

**To Restore Archived Chat:**

1. Go to **Archived** folder
2. Find conversation
3. Right-click ‚Üí **"Restore"**
4. Returns to main conversation list

## Temporary Chats

### Understanding Temporary Chats

**Temporary Chat Mode:**

```
‚ö†Ô∏è Temporary Chat Enabled

This conversation won't be saved to your chat history,
but any content you upload may still appear in the
content manager.
```

**Characteristics:**
- Not saved to conversation history
- Disappears after session ends
- Content may appear in content manager
- Useful for sensitive queries

### When to Use Temporary Chats

**Use Cases:**

**1. Confidential Information**
```
Example:
"Analyze this confidential contract [upload]"
"Review salary negotiation strategy"
"Process sensitive customer data"
```

**2. One-Time Queries**
```
Example:
"Quick fact check on competitor"
"Test a prompt idea"
"Personal advice"
```

**3. Experimental Prompts**
```
Example:
"Testing different writing styles"
"Comparing model outputs"
"Trying new approaches"
```

**4. Privacy-Sensitive Topics**
```
Example:
"Personal health questions"
"Financial planning scenarios"
"Private research"
```

### Limitations of Temporary Chats

**What's NOT Saved:**
- Conversation history
- Chat appears in sidebar
- Searchable archive

**What MAY Be Saved:**
- Uploaded files (in content manager)
- Credit usage logs
- System analytics (anonymous)

**Best Practices:**
- Use for truly temporary needs
- Don't rely on retrieving later
- Export important temporary chats before closing
- Check content manager for uploaded files

## Troubleshooting

### Common Issues

**Problem: Can't Find Old Conversation**
- **Solution**: Use search with keywords, check archived folder

**Problem: Export Failed**
- **Solution**: Try different format, check file size, refresh page

**Problem: Share Link Not Working**
- **Solution**: Check expiration date, verify permissions, regenerate link

**Problem: Temporary Chat Saved by Mistake**
- **Solution**: Delete from history, clear from content manager if needed

**Problem: Team Member Can't Access Shared Chat**
- **Solution**: Verify email address, check permissions, resend invite
