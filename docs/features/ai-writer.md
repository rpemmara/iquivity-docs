## Tips for Best Results

### Writing Effective Prompts

!!! tip "Prompt Formula"
    **Perfect Prompt = Context + Task + Format + Constraints + Examples**

**Example:**
```
Context: You're writing for a B2B SaaS company
Task: Create a blog post about cloud security
Format: 800 words, professional tone
Constraints: Must include 3 statistics and 2 expert quotes
Examples: Similar to articles on TechCrunch
```

### Iterative Refinement

1. **Start Broad** - Generate initial content
2. **Review** - Identify what works and what doesn't
3. **Refine Prompt** - Adjust instructions
4. **Regenerate** - Create improved version
5. **Polish** - Final human edits

### Credit Optimization

**Save Credits By:**
- Using appropriate length settings (don't generate long content if you need short)
- Writing better prompts (fewer regenerations needed)
- Using "Improve" on specific sections instead of regenerating entire documents
- Batch generating variations instead of individual requests

!!! note "Credit Costs"
    - Short text (< 300 words): 1-2 credits
    - Medium text (300-800 words): 3-5 credits
    - Long text (800-2000 words): 6-10 credits
    - Improvements/edits: 1-3 credits
