## Advanced Techniques

### Prompt Engineering

**Structure Your Prompts**

```
[Subject] + [Style] + [Details] + [Lighting] + [Quality] + [Artist Reference]

Example:
"Portrait of a woman" (subject)
+ "oil painting style" (style)
+ "wearing red dress, elegant pose" (details)
+ "soft natural lighting from window" (lighting)
+ "high detail, masterpiece" (quality)
+ "in the style of Rembrandt" (reference)
```

**Weight Keywords** ‚öñÔ∏è

Emphasize important elements:
```
"(((beautiful landscape))), mountains, lake, (highly detailed:1.5)"

() = slight emphasis
(()) = medium emphasis
((( ))) = strong emphasis
(keyword:1.5) = 1.5x weight
```

## Best Practices

### Do's ‚úÖ

- **Be Descriptive** - Include as many relevant details as possible
- **Specify Style** - Always mention the artistic style you want
- **Use References** - Mention artists, art styles, or platforms (e.g., "Artstation")
- **Test Iteratively** - Start simple, then add complexity
- **Save Good Prompts** - Keep a library of successful prompts
- **Use Negative Prompts** - Prevent common issues

### Don'ts ‚ùå

- **Don't Be Vague** - "A nice picture" won't give good results
- **Avoid Conflicting Descriptions** - Don't mix incompatible styles
- **Don't Over-Complicate** - Too many details can confuse the AI
- **Don't Expect Perfection** - AI has limitations, may need iterations
- **Don't Generate Copyrighted Content** - Respect intellectual property
