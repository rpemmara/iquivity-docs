### 2FA Setup Steps

**Step 1: Install Authenticator App**

Download and install one of these authenticator apps:
- **Google Authenticator** (iOS, Android)
- **Microsoft Authenticator** (iOS, Android)
- **Authy** (iOS, Android, Desktop)
- **1Password**
- **LastPass Authenticator**

**Step 2: Scan QR Code**

Open your authenticator app and scan the QR code displayed on the page.

**Alternative: Manual Entry**
If you can't scan the QR code:
1. Click the copy icon next to the manual key
2. Open your authenticator app
3. Choose "Enter key manually"
4. Paste the key (e.g., NHCIJFTSAR5WSMER)
5. Save the entry

**Step 3: Enter Verification Code**

1. Your authenticator app will display a 6-digit code
2. Enter this code on the 2FA setup page
3. Click **Enable 2FA**
4. 2FA is now activated!

!!! success "2FA Activated"
    Once enabled, you'll need to enter a code from your authenticator app every time you log in (unless you check "Trust this device").

### Disabling 2FA

If you need to disable 2FA:

1. Click your profile icon â†’ **2-Factor Auth.**
2. On the My Account page, under Security section, click **Deactivate 2FA**
3. Enter your current password
4. Enter current 2FA code from authenticator app
5. Confirm deactivation

!!! danger "Security Warning"
    Disabling 2FA makes your account less secure. Only disable if absolutely necessary.
