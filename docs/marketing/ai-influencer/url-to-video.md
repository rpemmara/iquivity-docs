# AI Influencer - URL to Video

Generate professional ad videos automatically from product URLs across multiple e-commerce platforms.

---

## Overview

URL to Video transforms product pages into engaging video advertisements using AI. Simply paste a product URL from supported platforms, and the AI analyzes product information, images, and details to create a professional marketing video.

!!! info "Automated Video Generation"
    AI automatically extracts product information, images, pricing, and features from URLs to create compelling video ads in minutes.

```

```

**Shopify:**
```
https://store-name.myshopify.com/products/product-name
https://custom-domain.com/products/product-name
```

**Etsy:**
```
https://www.etsy.com/listing/LISTING_ID/product-name
```

**eBay:**
```
https://www.ebay.com/itm/ITEM_ID
https://www.ebay.com/p/PRODUCT_ID
```

```
https://yourstore.com/product/product-name
https://yourstore.com/shop/product-name
```

```
https://play.google.com/store/apps/details?id=PACKAGE_NAME

```

## Troubleshooting

### Common Issues

**URL Not Analyzing:**
- **Cause:** Invalid URL, unsupported platform, page blocked
- **Solution:**
  - Verify URL is correct and accessible
  - Check if platform is supported
  - Try manual upload method
  - Ensure product page is public

**Poor Image Quality:**
- **Cause:** Low-resolution source images, compressed images
- **Solution:**
  - Use manual upload with high-res images
  - Contact seller for better images
  - Upscale images using AI tools first
  - Choose template that works with available quality

**Script Doesn't Match Product:**
- **Cause:** AI misunderstood product features, incomplete data
- **Solution:**
  - Manually edit script in Step 3
  - Add more product details in Step 2
  - Regenerate script with different tone
  - Write custom script manually

**Video Rendering Failed:**
- **Cause:** Processing error, timeout, invalid settings
- **Solution:**
  - Retry rendering
  - Check internet connection
  - Reduce video duration/resolution
  - Contact support if persists

**Wrong Aspect Ratio:**
- **Cause:** Incorrect platform selection in Step 4
- **Solution:**
  - Go back to Step 4
  - Select correct aspect ratio
  - Re-render video
  - Download multiple ratios for flexibility
