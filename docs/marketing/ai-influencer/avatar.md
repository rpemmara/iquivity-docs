## Manual Script Writing

### Script Writing Guidelines

**Script Best Practices:**

**Opening (0-5 seconds):**
```
Hook Examples:
- "Hey everyone! You won't believe this..."
- "I've been testing this for 2 weeks and..."
- "This changed everything for me..."
- "Let me show you something amazing..."
- "I need to tell you about this..."
```

**Body (5-50 seconds):**
```

- "Follow for more reviews like this!"
```

**Script Tips:**
- âœ… Write conversationally (how people speak)
- âœ… Use contractions (I'm, you're, it's)
- âœ… Include pauses with ellipses (...)
- âœ… Break into short sentences
- âœ… Add emphasis with capitalization sparingly
- âœ… Natural speech patterns (um, like, you know)

## Script Length Guidelines

### Optimal Durations

**15 Seconds (Quick):**
```
Word count: ~30-40 words
Use for:
- Product teasers
- Quick tips
- Announcements
- Flash sales
- Short hooks

Example length:
"Hey! Just got these amazing headphones and the
noise cancellation is incredible. 30-hour battery
too! Check them out - link below!"
```

**30 Seconds (Standard):**
```
Word count: ~70-80 words
Use for:
- Product reviews
- Quick demos
- Feature highlights
- Recommendations
- Most social content

Example length:
Full opening + 2-3 key points + CTA
```

**45 Seconds (Extended):**
```
Word count: ~100-120 words
Use for:
- Detailed reviews
- Comparisons
- How-to content
- Story-driven content
- In-depth explanations
```

**60 Seconds (Long):**
```
Word count: ~140-160 words
Use for:
- Complete tutorials
- Full product reviews
- Testimonials
- Brand stories
- Educational content
```

**90 Seconds (Maximum):**
```
Word count: ~200-220 words
Use for:
- Complex topics
- Multiple products
- Detailed storytelling
- Comprehensive guides
- Full presentations
```

## Troubleshooting

### Common Issues

**Poor Lip Sync:**
- **Cause:** Complex script, fast speech, punctuation issues
- **Solution:**
  - Simplify script structure
  - Add natural pauses (...)
  - Use shorter sentences
  - Adjust voice speed to "Slow"
  - Regenerate video

**Unnatural Voice:**
- **Cause:** Wrong accent selection, unnatural phrasing, speed issues
- **Solution:**
  - Select appropriate accent
  - Write more conversationally
  - Adjust speed and pitch
  - Use contractions (I'm, you're)
  - Add natural speech patterns

**Video Generation Failed:**
- **Cause:** Script too long, special characters, connection issue
- **Solution:**
  - Shorten script
  - Remove special characters/symbols
  - Check internet connection
  - Try different avatar
  - Contact support if persists

**Subtitles Not Appearing:**
- **Cause:** Subtitle option not enabled, rendering issue
- **Solution:**
  - Enable subtitles in settings
  - Regenerate video
  - Download with embedded subtitles
  - Check subtitle language matches script
