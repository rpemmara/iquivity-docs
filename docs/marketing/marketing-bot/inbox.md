# Marketing Bot - Inbox

Unified messaging interface for managing conversations across WhatsApp and Telegram in one centralized inbox.

---

## Overview

The Inbox provides a centralized hub for viewing and responding to messages from both WhatsApp and Telegram contacts. Manage all your conversations, track message history, and engage with your audience from a single interface.

!!! info "Unified Communication"
    All messages from WhatsApp and Telegram appear in one inbox, making it easy to manage customer conversations across platforms.

## Troubleshooting

### Common Issues

**Messages Not Loading:**
- **Cause:** Network connectivity, server issue
- **Solution:** Refresh page, check internet connection, retry

**Can't Send Messages:**
- **Cause:** Invalid number, platform issue, rate limit
- **Solution:** Verify contact number, check platform status, wait and retry

**Missing Conversations:**
- **Cause:** Archived, deleted, filter applied
- **Solution:** Check archived folder, review filters, search by contact

**Notifications Not Working:**
- **Cause:** Browser permissions, notification settings
- **Solution:** Enable browser notifications, check inbox settings
