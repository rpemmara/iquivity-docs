## Troubleshooting

### Common Issues

**Messages Not Sending:**
- **Cause:** Invalid API credentials, platform restrictions
- **Solution:** Check settings, verify tokens, ensure compliance

**Low Delivery Rates:**
- **Cause:** Invalid numbers, blocked contacts, network issues
- **Solution:** Clean contact list, verify numbers, test connectivity

**Campaign Not Scheduling:**
- **Cause:** Incorrect date format, past date selected
- **Solution:** Use date picker, verify time zone, check format

**Analytics Not Updating:**
- **Cause:** Cache delay, webhook issues
- **Solution:** Refresh page, check webhook configuration, wait for sync
