## Common Dashboard Actions

### Managing Connected Accounts

**Add New Account:**
1. Click **"Connect Accounts"**
2. Select platform (Facebook, Instagram, X, LinkedIn)
3. Click **"Add +"**
4. Authenticate with platform credentials
5. Grant necessary permissions

**View Account Status:**
- Active accounts display with green indicator
- Inactive accounts show reconnection prompt
- Account details include name and platform icon

### Creating Content

**Generate New Post:**
1. Click **"+ Generate New Post"** button
2. Follow the 6-step creation wizard:
   - **Step 1:** Platform - Select target social networks
   - **Step 2:** Company - Define brand information
   - **Step 3:** Campaign - Choose or create campaign
   - **Step 4:** Content - Generate AI content
   - **Step 5:** Review - Preview and edit
   - **Step 6:** Publish - Schedule or publish immediately

### Viewing Analytics

**Published Posts Analytics:**
- Displays line chart with monthly data
- Interactive controls for data exploration
- Platform-specific filtering
- Export capabilities for reporting

**Chart Insights:**
- **Peak Publishing Days** - Identify optimal posting times
- **Engagement Patterns** - Understand audience behavior
- **Content Performance** - Compare different post types
- **Platform Effectiveness** - Determine best-performing channels

## Troubleshooting

### Common Issues

**Dashboard Not Loading:**
- Check internet connection
- Clear browser cache
- Try different browser
- Contact support if issue persists

**Chart Data Not Displaying:**
- Ensure accounts are connected
- Verify posts have been published
- Check date range selection
- Refresh page

**Account Connection Failed:**
- Re-authenticate with platform
- Check platform API status
- Verify permissions granted
- Review account credentials
