# AI Social Media - Calendar

Visual scheduling interface for planning and managing social media content across all connected platforms.

---

## Overview

The Calendar view provides a comprehensive scheduling interface where you can visualize, plan, and manage your social media posts across all connected platforms. Schedule content days, weeks, or months in advance with an intuitive drag-and-drop interface.

!!! info "Visual Planning"
    The calendar displays all scheduled posts with color-coded platform indicators, making it easy to see your content distribution at a glance.

## Scheduling Posts

### Creating Scheduled Content

**Method 1: Calendar Click**

1. Click on desired date in calendar
2. Select time slot
3. Click **"+ Generate New Post"** button
4. Follow post creation wizard
5. Set publication time
6. Confirm schedule

**Method 2: Generate New Post Button**

1. Click **"+ Generate New Post"** in top-right
2. Complete post creation steps
3. In final step, choose "Schedule"
4. Select date and time from calendar picker
5. Confirm schedule

**Method 3: Drag and Drop (Week/Day View)**

1. Create draft post
2. Drag post card to desired time slot
3. Drop on calendar date/time
4. Confirm scheduling

## Managing Scheduled Posts

### Post Actions

**View Post Details:**
- Click on any scheduled post in calendar
- View full content, preview, and settings
- See all selected platforms
- Check scheduling status

**Edit Scheduled Post:**

1. Click on post in calendar
2. Select **"Edit"** option
3. Modify content, platforms, or time
4. Save changes
5. Updated post appears on calendar

**Reschedule Post:**

**Drag Method:**
1. Click and hold scheduled post
2. Drag to new date/time
3. Release to drop
4. Confirm new schedule

**Edit Method:**
1. Click on post
2. Select **"Reschedule"**
3. Choose new date and time
4. Save changes

**Delete Scheduled Post:**

1. Click on post in calendar
2. Select **"Delete"** or trash icon
3. Confirm deletion
4. Post removed from schedule

## Troubleshooting

### Common Issues

**Posts Not Appearing on Calendar:**
- **Cause:** Filter settings hiding content
- **Solution:** Check platform filters, reset to "All Platforms"

**Unable to Schedule Post:**
- **Cause:** Account connection expired
- **Solution:** Refresh account connection in Platforms page

**Schedule Conflicts:**
- **Cause:** Multiple posts at same time
- **Solution:** Use Day view to see conflicts, reschedule as needed

**Wrong Time Zone:**
- **Cause:** Time zone settings incorrect
- **Solution:** Check Settings > Time Zone, update to correct zone

**Drag and Drop Not Working:**
- **Cause:** Browser compatibility issue
- **Solution:** Try different browser, use Edit method instead
