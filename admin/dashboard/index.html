    Understanding the Metrics¶

    Revenue Metrics¶

    Earnings: - Total platform revenue - Includes subscriptions and one-time purchases - Excludes refunds and chargebacks

    Referral Payouts: - Commission owed to affiliates - Deducted from gross revenue - Paid on schedule (monthly/quarterly)

    Total Spending: - Operational costs - API usage fees - Infrastructure expenses

    User Metrics¶

    New Users: - First-time registrations - Excludes duplicate or test accounts - Week-over-week growth indicator

    AI Usage: - Total API calls to AI services - Across all tools and features - Measure of platform engagement

    Daily Visits: - Unique daily visitors - Active session count - Engagement metric

    Troubleshooting¶

    Dashboard Not Loading¶

    Issue: Dashboard metrics don't display

    Solutions: 1. Refresh the page (F5 or Cmd+R) 2. Clear browser cache 3. Check internet connection 4. Try different browser 5. Contact technical support

    Incorrect Data¶

    Issue: Metrics seem inaccurate

    Solutions: 1. Verify time period filter 2. Check for data sync delays 3. Compare with raw database 4. Wait for cache refresh (every 15 minutes) 5. Report data discrepancies

    Slow Performance¶

    Issue: Dashboard loads slowly

    Solutions: 1. Use shorter time periods (1D instead of 1Y) 2. Close unnecessary browser tabs 3. Check system resources 4. Use lighter browser 5. Contact support for optimization
