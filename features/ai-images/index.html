    AI Image Generation¶

    Create stunning visuals with AI-powered image technology. From creative artwork to professional product photography, iQuivity's AI image tools help you create professional visuals in seconds.

    Image Creation Tools:

    • AI Image Editor (NEW) - Edit and enhance existing images with AI
    • AI Image - Text-to-image generation with multiple models
    • AI Realtime Image - Real-time image generation with style presets
    • AI Product Shot - Professional product photography backgrounds
    • AI Chat Image - Conversational image generation
    • AI Vision - Image analysis and understanding

    Choose the tool that best fits your creative workflow and image needs.

    Use Cases: - Marketing and advertising visuals - Social media graphics - Website images - Creative artwork - Brand assets

    How to Use: 1. Navigate to Dashboard → AI IMAGE → Creative Suite 2. Enter your image prompt 3. Configure advanced settings 4. Generate professional-quality images

    AI Image Editor¶

    Features¶

    AI Image¶

    The core text-to-image generation tool with support for multiple AI models.

    Overview¶

    The AI Image tool uses advanced diffusion models to create unique, high-quality images from text descriptions. Whether you need marketing graphics, product mockups, illustrations, or creative art, the AI Image generator has you covered.

    AI Model Selection¶

    The AI Image tool supports multiple AI models for different image generation needs:

    Available Models: - Multiple model options for different styles and quality levels - Select based on your specific requirements - Each model has unique strengths

    How to Select Model: 1. Click the model selector dropdown 2. Browse available models 3. Read model descriptions 4. Select your preferred model 5. Click "Apply" to use the model

    Creating Images with AI Image¶

    Getting Started¶

    1. Navigate to AI Image
    2. Go to Dashboard → AI IMAGE → AI Image

    3. Enter the AI Image interface

    4. Choose Your Creation Method

    5. Text to Image - Create images from descriptions
    6. Configure model and settings
    7. Generate your image

    Step-by-Step Image Generation¶

    1. Enter Your Image Prompt¶

    Describe the image you want in detail. The more specific you are, the better the results.

    Example Prompts:

    Good Prompt:
"A futuristic city skyline at sunset, cyberpunk style, neon lights
reflecting on wet streets, flying cars in the background, high detail,
cinematic lighting, 4K quality, trending on artstation"

Basic Prompt:
"A city at sunset"

    Prompt Writing Tips

    Include these elements for best results:

    • Subject - What is the main focus?
    • Style - Photorealistic, artistic, cartoon, etc.
    • Details - Colors, lighting, atmosphere
    • Quality - High detail, 4K, professional
    • References - "Like a movie poster", "trending on artstation"

    2. Configure Image Settings¶

    Style Presets 🎨

    Choose from various artistic styles:

    • Photorealistic - Looks like a real photograph
    • Digital Art - Modern digital illustration style
    • Oil Painting - Classic painted appearance
    • Watercolor - Soft, flowing watercolor effect
    • Cartoon - Animated, illustrated style
    • Anime - Japanese animation style
    • 3D Render - Computer-generated 3D look
    • Pencil Sketch - Hand-drawn sketch appearance
    • Abstract - Non-representational art
    • Vintage - Retro, aged appearance
    • Minimalist - Simple, clean design
    • Cyberpunk - Futuristic, neon aesthetic

    Aspect Ratio 📐

    Select the dimensions that fit your needs:

    Ratio Dimensions Best For
    Square (1:1) 1024×1024 Social media, profile images
    Portrait (9:16) 768×1344 Stories, vertical content
    Portrait (2:3) 896×1344 Posters, prints
    Landscape (16:9) 1344×768 Presentations, headers
    Landscape (3:2) 1344×896 Photography, prints
    Widescreen (21:9) 1536×640 Banners, cover photos
    Custom Your choice Specific requirements

    Resolution Quality ✨

    • Standard (512×512) - Fast generation, good for previews
    • High (1024×1024) - Balanced quality and speed
    • Ultra (2048×2048) - Maximum detail, longer generation time

    Resolution Impact

    Higher resolutions consume more credits and take longer to generate. Start with Standard for testing, then use Ultra for final output.

    Number of Images

    Generate multiple variations at once (1-4 images per request):

    • See different interpretations
    • Choose the best result
    • A/B test variations
    • Save time on iterations

    3. Advanced Options¶

    Negative Prompt 🚫

    Specify what you DON'T want in the image:

    Example Negative Prompt:
"blurry, low quality, deformed, ugly, bad anatomy, watermark,
text, signature, out of frame, extra fingers"

    Seed Value 🎲

    • Use a specific number for reproducible results
    • Same seed + same prompt = same image
    • Leave blank for random generation
    • Share seeds to replicate results

    Guidance Scale 🎯

    Control how closely AI follows your prompt (1-20):

    Low (1-5)     ◄──────────●──────► High (15-20)
     Creative interpretation      Strict adherence
    • Low (1-7) - More creative freedom, unexpected results
    • Medium (7-12) - Balanced approach (recommended)
    • High (12-20) - Strict following of prompt details

    Steps 🔄

    Number of refinement iterations (20-150):

    • Quick (20-30) - Fast, good for testing
    • Standard (30-50) - Balanced quality
    • Detailed (50-100) - Higher quality
    • Ultra (100-150) - Maximum refinement

    Generation Time

    More steps = better quality but longer wait time and higher credit cost

    AI Vision¶

    Analyze, understand, and extract information from images using AI-powered computer vision.

    Overview¶

    AI Vision provides intelligent image analysis capabilities, allowing you to upload images and get detailed descriptions, object detection, text extraction, and insights.

    Features¶

    Image Analysis Capabilities:

    Visual Understanding: - Detailed image descriptions - Scene understanding - Object identification - Color analysis - Composition breakdown

    Text Recognition (OCR): - Extract text from images - Read handwriting (limited) - Document scanning - Sign and label reading - Screenshot text extraction

    Object Detection: - Identify objects in images - Count items - Location and bounding boxes - Category classification

    Content Analysis: - Brand logo detection - Facial detection (no identification) - Age/gender/emotion estimation - Content moderation flags - Quality assessment

    How to Use AI Vision¶

    Step 1: Upload Image 1. Navigate to Dashboard → AI IMAGE → AI Vision 2. Upload your image file or paste image URL 3. Supported formats: JPG, PNG, WEBP, GIF

    Step 2: Select Analysis Type

    Choose what you want to analyze:

    General Description - Get comprehensive image description - Scene understanding - Main elements identification - Overall composition analysis

    Text Extraction - Extract all text from image - Useful for: - Documents - Screenshots - Signs - Handwritten notes - Product labels

    Object Detection - Identify and locate objects - Get bounding boxes - Count detected items - Classification by category

    Ask Questions About Image - Ask specific questions about the image - "What color is the car?" - "How many people are in this photo?" - "What brand is visible in this image?" - "What is the main subject doing?"

    Step 3: Review Analysis Results

    Output Includes: - Detailed text descriptions - Extracted text (if OCR selected) - Detected objects list - Confidence scores - Additional insights

    Step 4: Export Results - Copy analysis to clipboard - Download as JSON - Save to workspace - Share with team

    Best Practices¶

    For Best Analysis Results:

    ✅ Image Quality: - Use clear, well-lit images - Avoid blurry or low-resolution photos - Ensure text is readable - Proper focus on subject

    ✅ Object Detection: - Objects clearly visible - Good lighting - Minimal occlusion - Appropriate distance

    Advanced Techniques¶

    Prompt Engineering¶

    Structure Your Prompts

    [Subject] + [Style] + [Details] + [Lighting] + [Quality] + [Artist Reference]

Example:
"Portrait of a woman" (subject)
+ "oil painting style" (style)
+ "wearing red dress, elegant pose" (details)
+ "soft natural lighting from window" (lighting)
+ "high detail, masterpiece" (quality)
+ "in the style of Rembrandt" (reference)

    Weight Keywords ⚖️

    Emphasize important elements: 

    "(((beautiful landscape))), mountains, lake, (highly detailed:1.5)"

() = slight emphasis
(()) = medium emphasis
((( ))) = strong emphasis
(keyword:1.5) = 1.5x weight

    Consistency Techniques¶

    Maintain Style Across Images 1. Use the same seed value 2. Keep style descriptors consistent 3. Reference previous successful prompts 4. Use style transfer (coming soon)

    Character Consistency (Advanced) 1. Describe character in detail 2. Save successful character prompts 3. Use consistent descriptors 4. Reference the same artistic style

    Credit Usage¶

    Image Generation Costs:

    Resolution Credits per Image
    Standard (512×512) 10 credits
    High (1024×1024) 25 credits
    Ultra (2048×2048) 50 credits
    Custom (varies) Calculated based on pixels

    Additional Features: - Image variations: 8-15 credits - Upscaling: 15-30 credits - Editing operations: 10-20 credits

    Save Credits

    • Start with Standard resolution for testing
    • Use Ultra only for final images
    • Generate multiple variations in one request (more efficient)
    • Use negative prompts to avoid unwanted results

    Best Practices¶

    Do's ✅¶

    • Be Descriptive - Include as many relevant details as possible
    • Specify Style - Always mention the artistic style you want
    • Use References - Mention artists, art styles, or platforms (e.g., "Artstation")
    • Test Iteratively - Start simple, then add complexity
    • Save Good Prompts - Keep a library of successful prompts
    • Use Negative Prompts - Prevent common issues

    Don'ts ❌¶

    • Don't Be Vague - "A nice picture" won't give good results
    • Avoid Conflicting Descriptions - Don't mix incompatible styles
    • Don't Over-Complicate - Too many details can confuse the AI
    • Don't Expect Perfection - AI has limitations, may need iterations
    • Don't Generate Copyrighted Content - Respect intellectual property

    Content Policy

    Images that violate our content policy will not be generated. This includes:

    • Explicit or adult content
    • Violence or gore
    • Illegal activities
    • Copyrighted characters (use inspired alternatives)
    • Hate symbols or offensive content
