    Using AI Chat Pro¶

    Learn how to effectively use AI Chat Pro for maximum productivity and get the best results from your AI conversations.

    Accessing AI Chat Pro¶

    Opening the Chat Interface¶

    From Dashboard:

    1. Log in to your iQuivity account
    2. Locate "AI Chat Pro" in the left-hand navigation menu
    3. Click to open the chat interface

    The chat interface will load with the Default AI Chat Bot ready to assist you.

    Selecting an AI Model¶

    Choosing the Right Model¶

    To Change AI Model:

    1. Click "AI Model: [Current Model]" button
    2. Browse available models
    3. Review each model's features:
    4. Name and description
    5. Credit cost (Unlimited or specific amount)
    6. Capabilities (Words, Images, etc.)
    7. Select desired model
    8. Click "Apply"

    Model Comparison¶

    Available Models by Provider:

    Always Up to Date

    All models are automatically updated to the latest versions from each provider. The specific model names may change as providers release newer versions.

    OpenAI Models¶

    Latest GPT Models - Type: Text and Multimodal (text + vision) - Tiers Available: Most advanced, turbo (faster), standard, lighter versions - Best For: General-purpose tasks, creative writing, image analysis, code generation - Speed: Varies by tier (from very fast to moderate) - Use When: Need versatile, well-rounded AI performance

    Strengths: - Industry-leading performance - Excellent multimodal capabilities - Strong at following complex instructions - Great for creative and analytical tasks

    Anthropic Models¶

    Latest Claude Models - Type: Text - Tiers Available: Most powerful, balanced, fastest - Best For: Deep analysis, long-form content, nuanced understanding, code generation - Speed: Varies by tier (from fastest to moderate) - Use When: Need sophisticated reasoning and detailed responses

    Strengths: - Superior analytical capabilities - Excellent for long conversations - Nuanced understanding of context - Strong safety and helpfulness

    Google Models¶

    Latest Gemini Models - Type: Multimodal (text, images, and more) - Tiers Available: Ultra (most capable), Pro (balanced), Flash (fastest) - Best For: Multimodal tasks, general-purpose work, quick responses - Speed: Varies by tier (from very fast to moderate) - Use When: Need multimodal capabilities or Google ecosystem integration

    Strengths: - Strong multimodal capabilities - Fast response times - Good balance of quality and speed - Integration with Google services

    Perplexity Models¶

    Perplexity AI - Type: Text with built-in web search - Best For: Research, current information, fact-checking, citations - Speed: Fast - Use When: Need real-time web data or cited sources

    Strengths: - Built-in real-time web search - Provides citations and sources - Excellent for research tasks - Up-to-date information

    DeepSeek Models¶

    DeepSeek Chat & Coder - Type: Text (with coding specialization) - Best For: Programming, code generation, debugging, technical tasks - Speed: Fast - Use When: Working on any coding or technical projects

    Strengths: - Specialized for programming tasks - Excellent code understanding - Strong technical reasoning - Cost-effective for development work

    Grok (X AI) Models¶

    Grok - Type: Text with real-time information - Best For: Conversational AI, current events, engaging responses - Speed: Fast - Use When: Need conversational responses or real-time information

    Strengths: - Engaging, personality-driven responses - Access to real-time information - Conversational and natural - Good for current events

    OpenRouter¶

    OpenRouter Gateway - Type: Multi-model access - Best For: Exploring different models, comparing outputs - Speed: Varies by selected model - Use When: Want access to various models through one interface

    Strengths: - Access to multiple AI providers - Regularly updated model catalog - Flexibility to try different models - Unified interface

    Model Selection Strategy¶

    For Different Tasks:

    Task Type Recommended Provider Why
    Quick Questions Anthropic (fastest tier), OpenAI (lighter), Google (Flash) Fastest responses
    Content Writing OpenAI, Anthropic, Google High quality, creative
    Code Generation DeepSeek, OpenAI, Anthropic Technical accuracy, specialized
    Code Debugging DeepSeek, Anthropic, OpenAI Deep code understanding
    Data Analysis Anthropic, OpenAI, Google Deep reasoning
    Research Perplexity Real-time web access with citations
    Casual Chat Grok, Anthropic, OpenAI Conversational, engaging
    Image Analysis OpenAI, Google Vision capabilities
    Long Documents Anthropic, OpenAI Large context window
    Current Events Perplexity, Grok Real-time information
    Model Exploration OpenRouter Access to variety
    General Purpose OpenAI, Anthropic, Google Balanced performance

    Model Selection Tips

    • Try multiple providers for the same task to compare results
    • Choose by tier within each provider (fastest, balanced, most powerful)
    • Models update automatically - you always get the latest versions
    • Check credit costs for each model before extensive use

    Starting a Conversation¶

    Using Prompt Suggestions¶

    Quick Start with Prompts:

    When you open a chatbot, you'll see suggested prompts:

    [Translate This Book]
[Generate a cute panda image]
[Plan a 3 day trip to Rome]
[Pick an outfit]
[How can I learn coding?]
[Experience Tokyo]

    To Use: - Click any suggestion to start a conversation - The prompt will be sent automatically - AI will respond based on the selected model

    Typing Your Own Message¶

    To Start a Custom Conversation:

    1. Click in the message field at the bottom
    2. Type your question or request
    3. Press Enter or click the Send button (‚ñ∂)

    Example Messages: 

    "Explain quantum computing in simple terms"
"Write a professional email to decline a job offer"
"Debug this Python code: [paste code]"
"Create a workout plan for beginners"
"Summarize the key points of [topic]"

    Best Practices for Prompts¶

    Be Specific: 

    ‚ùå Vague: "Tell me about marketing"
‚úÖ Specific: "Explain the 5 key principles of content marketing for B2B SaaS companies"

    Provide Context: 

    ‚ùå No Context: "Write an email"
‚úÖ With Context: "Write a professional follow-up email to a client who hasn't responded to my proposal in 2 weeks. Tone should be friendly but assertive."

    Break Down Complex Tasks: 

    Step 1: "I need to create a marketing strategy. First, help me identify my target audience."
[Get response]
Step 2: "Now, based on that audience, what channels should I focus on?"
[Get response]
Step 3: "Great! Now help me create a content calendar for the first month."

    Brand Voice¶

    Customize AI Responses:

    1. Click the Brand Voice icon (üòé) in the input area

    How It Works: - AI adapts tone and style to match your brand - Responses aligned with your company voice - Consistent messaging across all outputs

    Use Cases: - Generate marketing copy - Create customer communications - Write product descriptions - Draft social media posts - Maintain brand consistency

"Our solution delivers measurable time savings through
advanced automation capabilities."

    4. Ask the same question
    5. Compare answers

    Use When: - Need different perspectives - Verify information - Choose best output - Test different provider strengths

    Example Comparison:

OpenAI: Technical, comprehensive answer
Anthropic: Nuanced, analytical approach

    Troubleshooting¶

    Common Issues¶

    Problem: AI Not Responding - Cause: Network issue or model overloaded - Solution: Refresh page, try different model

    Problem: Response Quality Poor - Cause: Vague prompt or model not suited for task - Solution: Be more specific in prompt, try different provider or higher-tier model

    Problem: Conversation Lost Context - Cause: Too many turns or model limitation - Solution: Summarize previous context in next message

    Problem: File Upload Failed - Cause: File too large or unsupported format - Solution: Compress file, convert to supported format

    Problem: Credits Running Low - Cause: Using credit-consuming models/features - Solution: Switch to unlimited credit models, upgrade plan
