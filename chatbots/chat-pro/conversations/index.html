    Managing Conversations¶

    Learn how to organize, search, export, and share your AI Chat Pro conversations for maximum productivity.

    Conversation History¶

    Accessing Your Conversations¶

    View All Conversations:

    1. Open AI Chat Pro from the main navigation
    2. Your current and recent conversations appear automatically
    3. Conversation history shows:
    4. Recent chats
    5. Chat titles (auto-generated or custom)
    6. Last message preview
    7. Timestamp

    Organizing Conversations¶

    Creating New Conversations¶

    To Start a New Chat:

    1. Click the "+" (New Chat) button
    2. Select a chatbot (Default or Expert)
    3. Choose AI model
    4. Start typing

    When to Create New Chats: - Different topics/projects - Separate work contexts - Testing different models - Privacy/organization

    Switching Between Conversations¶

    To Switch Chats:

    1. Click on any chatbot in the sidebar
    2. Previous conversation loads instantly
    3. Context from that conversation preserved
    4. Continue where you left off

    Benefits: - Keep multiple projects organized - Maintain separate contexts - Easy reference to past work - Quick topic switching

    Naming Conversations¶

    Auto-Generated Titles: - AI automatically titles conversations - Based on initial topic/question - Updates as conversation evolves

    Custom Titles: 1. Click on conversation name (if editable) 2. Enter custom title 3. Save

    Example Titles: 

    "Marketing Strategy Q1 2025"
"Python Code Debugging Session"
"Content Ideas for Blog"
"Budget Planning Discussion"
"Email Template Creation"

    Searching Conversations¶

    Finding Past Chats¶

    Search Features:

    1. Click search box at top of sidebar
    2. Type keywords:
    3. Chat title
    4. Message content
    5. Topic
    6. Date
    7. Results appear instantly

    Example Searches: 

    "marketing" → All marketing-related chats
"Python code" → Code debugging conversations
"last week" → Recent conversations
"client email" → Email drafting sessions

    Filtering Conversations¶

    Filter Options:

    By Chatbot: - Show only Finance Expert chats - Filter by Nutritionist conversations - View all Default AI chats

    By Date: - Today - Last 7 days - Last 30 days - Custom date range

    By Status: - Active conversations - Archived chats - Shared conversations - Temporary chats (if saved)

    Exporting Conversations¶

    Export Options¶

    To Export a Conversation:

    1. Open the conversation you want to export
    2. Click More Options (⋯) at top-right
    3. Select "Export"
    4. Choose format:
    5. Text (.txt) - Plain text format
    6. PDF (.pdf) - Formatted document
    7. Markdown (.md) - Markdown format
    8. JSON - Raw data format
    9. Click "Download"

    Export Formats Explained¶

    Text (.txt) 

    Conversation Export
Date: December 6, 2025
Chatbot: Default AI Chat Bot
Model: GPT-4o

User: Explain machine learning in simple terms

AI: Machine learning is a type of artificial intelligence
that allows computers to learn and improve from experience
without being explicitly programmed...

User: Can you give me a practical example?

AI: Sure! A great example is email spam filters...

    Best For: - Simple archiving - Copying to other apps - Quick reference - Email sharing

    PDF (.pdf) - Professional formatting - Headers and structure - Syntax highlighting for code - Images preserved - Ready to print

    Best For: - Presentations - Client deliverables - Documentation - Professional sharing - Archiving

    Markdown (.md) - Structured formatting - Code blocks with syntax highlighting - Headers and lists formatted - Compatible with documentation tools

    Best For: - Technical documentation - GitHub/GitLab wikis - Knowledge bases - Developer collaboration

    JSON - Raw conversation data - Includes metadata - Timestamps - Model information - Message structure

    Best For: - Data analysis - Custom processing - API integration - Backup purposes

    Bulk Export¶

    Export Multiple Conversations:

    1. Go to conversation management
    2. Select multiple chats (checkbox)
    3. Click "Export Selected"
    4. Choose format
    5. Download as ZIP file

    Use Cases: - Weekly/monthly backups - Project archives - Team knowledge sharing - Compliance/record keeping

    Sharing Conversations¶

    Share with Team Members¶

    To Share a Conversation:

    1. Open the conversation
    2. Click More Options (⋯)
    3. Select "Share"
    4. Choose sharing method

    Sharing Methods:

    1. Share Link¶

    Generate shareable link:
https://app.iquivity.ai/chat/share/abc123xyz

Options:

☐ Password protected
☐ Expires after: [7 days ▼]

    Best For: - Quick sharing - External collaborators - Temporary access

    2. Share with Team¶

    Select team members:

☑ Sarah Johnson (sarah@company.com)
☐ Mike Chen (mike@company.com)

Permissions:
● View only
○ Can comment
○ Can edit

    Best For: - Internal collaboration - Team projects - Controlled access

    3. Email Conversation¶

    Send to: [email@example.com]
Subject: [Shared Chat: Marketing Strategy]

Message:
[Optional message to recipient]

[Send Email]

    Best For: - Client communication - Executive summaries - Remote team members

    Managing Shared Conversations¶

    Permissions Levels:

    View Only: - Can read entire conversation - Cannot add messages - Cannot export - Cannot reshare

    Can Comment: - View full conversation - Add comments/notes - Cannot modify messages - Can export

    Can Edit: - Full access to conversation - Can add messages - Can continue chat - Can export and reshare

    View Share Activity:

    Features: - See who has access - Track views and engagement - Revoke access anytime - Update permissions

    Revoking Access¶

    To Remove Access:

    1. Go to Shared Conversations
    2. Find the shared conversation
    3. Click "Manage Access"
    4. Select person/link
    5. Click "Revoke Access"
    6. Confirm

    When to Revoke: - Project completed - Team member left - Link expired - Security concern

    Conversation Management¶

    Archiving Conversations¶

    To Archive:

    1. Right-click on conversation (or use menu)
    2. Select "Archive"
    3. Conversation moves to archive

    Archived Conversations: - Removed from main list - Still searchable - Can be restored - Preserved indefinitely

    When to Archive: - Completed projects - Old conversations - Seasonal content - Reference material

    Deleting Conversations¶

    To Delete:

    1. Right-click on conversation
    2. Select "Delete"
    3. Confirm deletion

    Deletion is Permanent

    Deleted conversations cannot be recovered. Export important chats before deleting.

    When to Delete: - Test conversations - Duplicate chats - Sensitive information (after use) - Temporary experiments

    Restoring Conversations¶

    To Restore Archived Chat:

    1. Go to Archived folder
    2. Find conversation
    3. Right-click → "Restore"
    4. Returns to main conversation list

    Temporary Chats¶

    Understanding Temporary Chats¶

    Temporary Chat Mode:

    ⚠️ Temporary Chat Enabled

This conversation won't be saved to your chat history,
but any content you upload may still appear in the
content manager.

    Characteristics: - Not saved to conversation history - Disappears after session ends - Content may appear in content manager - Useful for sensitive queries

    When to Use Temporary Chats¶

    Use Cases:

    1. Confidential Information 

    Example:
"Analyze this confidential contract [upload]"
"Review salary negotiation strategy"
"Process sensitive customer data"

    2. One-Time Queries 

    Example:
"Quick fact check on competitor"
"Test a prompt idea"
"Personal advice"

    3. Experimental Prompts 

    Example:
"Testing different writing styles"
"Comparing model outputs"
"Trying new approaches"

    4. Privacy-Sensitive Topics 

    Example:
"Personal health questions"
"Financial planning scenarios"
"Private research"

    Limitations of Temporary Chats¶

    What's NOT Saved: - Conversation history - Chat appears in sidebar - Searchable archive

    What MAY Be Saved: - Uploaded files (in content manager) - Credit usage logs - System analytics (anonymous)

    Best Practices: - Use for truly temporary needs - Don't rely on retrieving later - Export important temporary chats before closing - Check content manager for uploaded files

    Bulk Actions:

    Select Multiple Conversations: - Archive all completed projects - Export multiple chats at once - Share bundle with team - Delete temporary experiments

    Troubleshooting¶

    Common Issues¶

    Problem: Can't Find Old Conversation - Solution: Use search with keywords, check archived folder

    Problem: Export Failed - Solution: Try different format, check file size, refresh page

    Problem: Share Link Not Working - Solution: Check expiration date, verify permissions, regenerate link

    Problem: Temporary Chat Saved by Mistake - Solution: Delete from history, clear from content manager if needed

    Problem: Team Member Can't Access Shared Chat - Solution: Verify email address, check permissions, resend invite

    Privacy & Security¶

    Data Protection¶

    Your Conversations: - Encrypted in transit and at rest - Stored securely in your iQuivity account - Access controlled by you - Not used for AI training (check provider terms)

    Sharing Security: - Password-protected links available - Expiration dates on shares - Revocable access anytime - Audit trail of all shares

    Best Practices: - Don't share sensitive data via public links - Review shared access regularly - Use temporary chats for confidential topics - Export and delete when no longer needed
