    Dashboard Overview¶

    Your dashboard is the central hub for all iQuivity AI activities. Learn how to navigate and make the most of your workspace.

    Dashboard Layout¶

    When you first log in, you'll see your personalized dashboard with several key sections designed to help you work efficiently.

    Quick Actions¶

    The Quick Actions panel provides instant access to your most common tasks.

    Available Actions¶

    New Document - Start a new AI-generated text document - Opens the AI Writer with a blank canvas - Choose from templates or custom prompts

    Create Image - Launch the AI Image Generator - Start with a text prompt or template - Generate visual content instantly

    Build Chatbot - Create a new chatbot from scratch - Configure and train your AI assistant - Deploy to your website

    Upload File - Add files to your workspace - Supported formats: documents, images, videos, and more - Drag and drop or browse to upload

    Keyboard Shortcuts

    Speed up your workflow with keyboard shortcuts:

    • ⌘/Ctrl + N - New Document
    • ⌘/Ctrl + U - Upload File
    • ⌘/Ctrl + K - Search everything

    Recent Activity¶

    Stay on top of your work with the Recent Activity section, which displays:

    Activity Types¶

    Recent Content - Last 10 documents you created or edited - Timestamp of last modification - Quick preview and access

    Ongoing Projects - In-progress documents and videos - Draft chatbots - Incomplete workbooks

    Collaboration Updates - Team member edits and comments - Shared documents that have been modified - @mentions and notifications

    File Changes - Recently uploaded files - Modified or shared files - Deleted items (recoverable from trash)

    Activity Timeline

    Your activity feed shows items from the last 30 days. Older items can be found in their respective sections (Files, AI Writer, etc.)

    Usage Statistics¶

    Monitor your account resources and usage in real-time.

    Credits Remaining¶

    What Are Credits? - Credits power AI operations on the platform - Different operations consume different amounts - Your plan includes a monthly credit allocation

    Low Credit Alert

    When you have less than 10% of your credits remaining, you'll see a warning banner. Consider upgrading your plan or purchasing additional credits.

    Storage Used¶

    Storage Breakdown - Total Storage: See your storage limit based on your plan - Used Space: Current storage consumption - Available: Remaining storage capacity

    Storage by Type - Documents: 30% - Images: 45% - Videos: 20% - Other: 5%

    Free Up Space

    Empty your trash to free up storage. Deleted items are kept for 30 days before permanent deletion.

    Monthly Usage¶

    Usage Metrics - Total AI Requests: Number of AI operations this month - Content Created: Count of documents, images, videos - Chatbot Conversations: Number of chatbot interactions - API Calls: API usage (if applicable)

    Plan Details¶

    Current Plan Information - Plan name (Free, Starter, Professional, Enterprise) - Renewal date - Billing amount - Quick link to upgrade/manage subscription

    Navigation Menu¶

    Your main navigation menu provides access to all platform features.

    Core Features¶

    Dashboard 🏠 - Return to home screen - View overview and statistics

    AI Writer ✍️ - Text content generation - Blog posts, articles, marketing copy - Access to writing templates - Learn more →

    AI Images 🎨 - Image generation from text - Edit and enhance images - Image gallery management - Learn more →

    AI Videos 🎬 - Video creation tools - URL to video conversion - Viral clip generation - Learn more →

    Chatbots 🤖 - Create and manage chatbots - Training and customization - Analytics and conversations - Learn more →

    Analytics 📊 - Usage reports and insights - Performance metrics - Team activity (if applicable)

    Settings ⚙️ - Account preferences - Security settings - Billing and subscription - API access - Learn more →

    Navigation Tips¶

    Search Functionality

    Use the search bar at the top of the dashboard to quickly find:

    • Documents and files
    • Templates
    • Chatbots
    • Settings and features

    Search Shortcuts - Type / to focus search bar - Use filters: type:image, date:today, tag:marketing - Recent searches are saved for quick access

    Customizing Your Dashboard¶

    Widget Arrangement¶

    Reorder Widgets (Coming Soon) - Drag and drop dashboard sections - Show or hide specific widgets - Create a layout that works for you

    Dashboard Preferences¶

    Access via Settings → Dashboard Preferences

    Customize Display - Choose compact or expanded view - Set default landing page - Configure activity feed filters - Show/hide usage statistics

    Theme Options - Light mode (default) - Dark mode - Auto (system preference)

    Personalization

    Your dashboard preferences are saved per account and will be consistent across all devices.

    Mobile Dashboard¶

    The dashboard is fully responsive and optimized for mobile devices.

    Mobile Features¶

    Simplified Layout - Touch-optimized interface - Swipe navigation - Mobile-friendly menus

    Quick Actions - Access via floating action button (FAB) - Create content on the go - Upload photos directly from camera

    Offline Mode (Limited) - View cached documents - Access recent activity - Sync when connection is restored

    Dashboard Shortcuts¶

    Speed up your workflow with these helpful shortcuts:

    Action Mac Windows
    New Document ⌘ + N Ctrl + N
    Search ⌘ + K Ctrl + K
    Upload File ⌘ + U Ctrl + U
    Settings ⌘ + , Ctrl + ,
    Help ⌘ + ? Ctrl + ?

    Pro Tip

    Master these shortcuts to navigate the platform 2-3x faster! Press ⌘ + ? (or Ctrl + ?) to see all available shortcuts.

    Getting Help from Dashboard¶

    Help Resources¶

    Access help directly from your dashboard:

    Help Button (?) - Located in top-right corner - Quick links to documentation - Video tutorials - Live chat support

    Guided Tours - First-time user walkthrough - Feature-specific tours - Can be restarted anytime

    Tooltips - Hover over any icon or button - See brief explanations - Learn features as you go

    Support Access

    Premium plan users have access to live chat support directly from the dashboard. Free and Starter plan users can submit support tickets.
